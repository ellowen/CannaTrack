Politica de Privacidad

Ultima actualizacion: Abril 2026

Que datos recolectamos

  • Cuenta: tu direccion de email para autenticacion.
  • Cultivo: datos que vos ingresás: nombre de plantas, genetica, fechas, notas, fotos.
  • Uso: diagnosticos IA solicitados (conteo mensual para respetar los limites del plan).
  • Dispositivo: token de notificaciones push (solo si das permiso).

Como usamos los datos

  • Para mostrarte tu historial de cultivo y tareas del dia.
  • Para enviar recordatorios de riego/nutricion (si activas las notificaciones).
  • Para el diagnostico IA: las fotos se envían a la API de Anthropic y no se almacenan de forma permanente en sus servidores.
  • Para controlar los limites mensuales del plan Free/Pro.

No vendemos tus datos

CannaTrack no vende, alquila ni comparte datos personales con terceros con fines comerciales. Los unicos proveedores que reciben datos son los necesarios para el funcionamiento del servicio (Supabase para base de datos/autenticacion, Anthropic para analisis de fotos, RevenueCat para suscripciones, Expo para notificaciones push).

Fotos de plantas

Las fotos que subis al diario se almacenan en servidores seguros (Supabase Storage). Solo vos podes verlas. Las fotos enviadas al diagnostico IA son procesadas por Anthropic y no se almacenan de forma permanente.

Seguridad

Todos los datos se transmiten con HTTPS. El acceso a la base de datos esta protegido por Row Level Security: ningún usuario puede ver datos de otro.

Tus derechos

Podes solicitar la eliminacion de tu cuenta y todos tus datos en cualquier momento escribiendo a privacidad@cannatrack.app.
