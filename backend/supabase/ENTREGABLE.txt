NOTAS IMPORTANTES
================================================================================

1. Las migrations son idempotentes (IF NOT EXISTS)
   -> Seguro ejecutar varias veces

2. RLS está habilitado en TODAS las tablas
   -> Usuarios NO pueden ver datos de otros
   -> nutrition_* es público (lectura)

3. Triggers automáticos:
   - handle_new_user: Se dispara al registrarse
   - handle_task_completion: Se dispara al actualizar completed=true
   - handle_photo_upload: Preparada para Edge Function

4. Storage folder structure:
   /{user_id}/{plant_id}/filename.jpg
   RLS verifica que user_id sea auth.uid()

5. Tabla REVEGETAR es la default en plan Free
   - 14 semanas (6 vege + 8 flora)
   - 20 productos listos para usar
   - EC/PH ranges por etapa

6. Sistema de niveles es escalable
   - Funciones cálculan automáticamente
   - Fácil agregar más acciones de XP
   - Bonus system: cada 5, 10, 15... días de racha
