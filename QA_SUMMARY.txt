5 CRITICAL/HIGH-SEVERITY BUGS FOUND
================================================================================

1. MISSING awardXP CALL [CRITICAL]
   Location: src/components/CompleteTaskSheet.tsx (line 194)
   Impact: User sees +25 XP animation, but XP never synced to database
           On reload, XP is gone. Streak never increments.
   Fix Time: 30 minutes

2. useInitSync OVERWRITES LOCAL DATA [CRITICAL]
   Location: src/hooks/useInitSync.ts (line 26)
   Impact: If network request fails, local plants/tasks are cleared
           User loses offline changes when app starts
   Fix Time: 20 minutes

3. RACE CONDITION IN STREAK UPDATE [HIGH]
   Location: src/lib/xp.ts (line 30)
   Impact: User completes 2 tasks rapidly → streak increments once
           Misses 7-day and 30-day milestone bonuses
   Fix Time: 45 minutes

4. NETWORK POLLING MEMORY LEAK [HIGH]
   Location: src/lib/network.ts (line 34-44)
   Impact: Network check every 5 sec accumulates over time
           Battery drain, memory growth, stale listeners
   Fix Time: 30 minutes

5. ASYNCSTORAGE CORRUPTION NOT CAUGHT [MEDIUM]
   Location: src/lib/storage.ts (line 7-26)
   Impact: Invalid dates cause silent failures during sync
           "Invalid Date" sent to Supabase, sync fails
   Fix Time: 25 minutes
